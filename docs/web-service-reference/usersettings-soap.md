# <a name="usersettings-soap"></a>UserSettings (SOAP)

L’élément **UserSettings** représente une collection de paramètres utilisateur. 
  
```XML
<UserSettings>
    <UserSetting/>
</UserSettings>
```

 **UserSettings**
