# <a name="getfederationinformationrequestmessage-soap"></a>GetFederationInformationRequestMessage (SOAP)

L’élément **GetFederationInformationRequestMessage** prépare un appel au serveur pour demander des données de configuration pour le service de jeton de sécurité (STS). 
  
```XML
<GetFederationInformationRequestMessage>
   <Request/>
</GetFederationInformationRequestMessage>

```
