# <a name="arrayofuserresponse-soap"></a>ArrayOfUserResponse (SOAP)

L’élément **ArrayOfUserResponse** contient un tableau d’éléments de [Réponse utilisateur (SOAP)](userresponse-soap.md) . 
  
```XML
<ArrayOfUserResponse>
   <UserResponse/>
</ArrayOfUserResponse>
```
