# <a name="searchmailboxes"></a>SearchMailboxes

L’élément **SearchMailboxes** indique le début de la demande **SearchMailboxes** . 
  
```XML
<SearchMailboxes>
   <SearchQueries/>
   <ResultType/>
   <PreviewItemResponseShape/>
   <SortBy/>
   <Language/>
   <Deduplication/>
   <PageSize/>
   <PageItemReference/>
   <PageDirection/>
</SearchMailboxes>
```
