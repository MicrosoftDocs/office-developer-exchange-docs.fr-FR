# <a name="getfederationinformationrequest-soap"></a>GetFederationInformationRequest (SOAP)

L’élément **GetFederationInformationRequest** représente les paramètres d’un appel à l’opération [GetFederationInformation (SOAP)](getfederationinformation-operation-soap.md).
  
```XML
<GetFederationInformationRequest>
   <Domain/>
</GetFederationInformationRequest>
```

**GetFederationInformationRequest**
