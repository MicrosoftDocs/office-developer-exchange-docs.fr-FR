# <a name="addnewteluricontacttogroupresponse"></a>AddNewTelUriContactToGroupResponse

L’élément **AddNewTelUriContactToGroupResponse** spécifie les données de résultat de l’opération WSDL **AddNewTelUriContactToGroup** . 
  
```XML
<AddNewTelUriContactToGroupResponse>
   <Persona/>
   <MessageText/>
   <ResponseCode/>
   <DescriptiveLinkKey/>
</AddNewTelUriContactToGroupResponse>
```

 **AddNewTelUriContactToGroupResponseMessageType**
## <a name="attributes-and-elements"></a>Attributs et éléments

Les sections suivantes décrivent les attributs, éléments enfants et éléments parents.
  
### <a name="attributes"></a>Attributs

Aucun.
  
### <a name="child-elements"></a>Éléments enfants

[Personnage](persona.md) | [MessageText](messagetext.md) | [ResponseCode](responsecode.md) | [DescriptiveLinkKey](descriptivelinkkey.md)
  
### <a name="parent-elements"></a>Éléments parents

Aucun.
  
## <a name="remarks"></a>Remarques

Cet élément est une nouveauté d'Exchange Server 2013.
  
Le schéma qui décrit cet élément se trouve dans le répertoire virtuel IIS qui héberge les services web Exchange.
