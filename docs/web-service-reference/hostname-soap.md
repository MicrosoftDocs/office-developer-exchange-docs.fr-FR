# <a name="hostname-soap"></a>Nom d’hôte (SOAP)

L’élément **nom d’hôte** représente la partie de nom d’hôte du nom complet de l’ordinateur de l’ordinateur. 
  
```XML
<Hostname/>
```

 **string**
## <a name="attributes-and-elements"></a>Attributs et éléments

Les sections suivantes décrivent les attributs, éléments enfants et éléments parents.
  
### <a name="attributes"></a>Attributs

Aucun.
  
### <a name="child-elements"></a>Éléments enfants

Aucun.
  
### <a name="parent-elements"></a>Éléments parents

|**Élément**|**Description**|
|:-----|:-----|
|[ProtocolConnection (SOAP)](protocolconnection-soap.md) <br/> |Représente le protocole de connexion du client Web server.  <br/> |
   
## <a name="text-value"></a>Valeur de texte

La valeur de texte de l’élément de **nom d’hôte** est la partie de nom d’hôte du nom complet de l’ordinateur de l’ordinateur.
