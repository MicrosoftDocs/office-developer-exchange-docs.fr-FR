# <a name="alternatemailboxes-soap"></a>AlternateMailboxes (SOAP)

L’élément **AlternateMailboxes** représente une collection d’autres boîtes aux lettres. 
  
```XML
<AlternateMailboxes>
   <AlternateMailbox/>
</AlternateMailboxes>
```

 **AlternateMailboxes**
