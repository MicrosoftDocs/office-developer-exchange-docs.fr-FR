# <a name="mailtipsaccessenabled-soap"></a>MailTipsAccessEnabled (SOAP)

L’élément **MailTipsAccessEnabled** représente l’indicateur **MailTipsAccessEnabled ()** . L’élément **MailTipsAccessEnabled** est réservé à un usage interne. Cet élément n’est pas utilisé par les clients. 
  
```XML
<MailTipsAccessEnabled>true | false</ MailTipsAccessEnabled>
```

 **Boolean**
## <a name="attributes-and-elements"></a>Attributs et éléments

Les sections suivantes décrivent les attributs, les éléments enfants et les éléments parents.
