# <a name="phonenumberattributedvalue"></a>PhoneNumberAttributedValue

L’élément **PhoneNumberAttributedValue** spécifie une instance d’un tableau des numéros de téléphone et leurs attributions associées. 
  
```XML
<PhoneNumberAttributedValue>
   <Value/>
   <Attributions/>
</PhoneNumberAttributedValue>
```

 **PhoneNumberAttributedValueType**
## <a name="attributes-and-elements"></a>Attributs et éléments

Les sections suivantes décrivent les attributs, éléments enfants et éléments parents.
  
### <a name="attributes"></a>Attributs

Aucun.
