# <a name="usersetting-soap"></a>UserSetting (SOAP)

L’élément [UserSetting (SOAP)](usersetting-soap.md) représente un paramètre d’utilisateur unique. 
  
```XML
<UserSetting>
    <Name />
</UserSetting>
```

 **UserSetting**
## <a name="attributes-and-elements"></a>Attributs et éléments

Les sections suivantes décrivent les attributs, éléments enfants et éléments parents.
  
### <a name="attributes"></a>Attributs

Aucun.
  
### <a name="child-elements"></a>Éléments enfants

|**Élément**|**Description**|
|:-----|:-----|
|[Nom (SOAP)](name-soap.md) <br/> |Représente le nom du paramètre de l’utilisateur.  <br/> |
   
### <a name="parent-elements"></a>Éléments parents

|**Élément**|**Description**|
|:-----|:-----|
|[UserSettings (SOAP)](usersettings-soap.md) <br/> |Représente une collection de paramètres utilisateur.  <br/> |
