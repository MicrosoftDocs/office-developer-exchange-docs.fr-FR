# <a name="displayname-soap"></a>DisplayName (SOAP)

L’élément **DisplayName** représente le nom complet. 
  
```XML
<DisplayName/>
```

 **string**
## <a name="attributes-and-elements"></a>Attributs et éléments

Les sections suivantes décrivent les attributs, éléments enfants et éléments parents.
  
### <a name="attributes"></a>Attributs

Aucun.
  
### <a name="child-elements"></a>Éléments enfants

Aucun.
  
### <a name="parent-elements"></a>Éléments parents

|**Élément**|**Description**|
|:-----|:-----|
|[AlternateMailbox (SOAP)](alternatemailbox-soap.md) <br/> |Représente une autre boîte aux lettres.  <br/> |
|[DocumentSharingLocation (SOAP)](documentsharinglocation-soap.md) <br/> |Représente les informations d’emplacement et les métadonnées pour un emplacement de partage de document.  <br/> |
   
## <a name="text-value"></a>Valeur de texte

La valeur de texte de l’élément **DisplayName** est le nom complet d’un paramètre.
