# <a name="minorversion-soap"></a>MinorVersion (SOAP)

L’élément **MinorVersion** représente le numéro de version mineure pour le serveur. 
  
```XML
<MinorVersion/>
```

 **int**
## <a name="attributes-and-elements"></a>Attributs et éléments

Les sections suivantes décrivent les attributs, les éléments enfants et les éléments parents.
  
### <a name="attributes"></a>Attributs

Aucune.
  
### <a name="child-elements"></a>Éléments enfants

Aucun.
  
### <a name="parent-elements"></a>Éléments parents

|**Élément**|**Description**|
|:-----|:-----|
|[ServerVersionInfo (SOAP)](serverversioninfo-soap.md) <br/> |Contient la version du serveur qui a traité la demande.  <br/> |
   
## <a name="text-value"></a>Valeur texte

Valeur entière qui représente le numéro de version mineure du serveur qui a traité la demande.
