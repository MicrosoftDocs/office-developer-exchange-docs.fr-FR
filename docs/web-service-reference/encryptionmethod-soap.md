# <a name="encryptionmethod-soap"></a>EncryptionMethod (SOAP)

L’élément **EncryptionMethod** représente la méthode de chiffrement qui est utilisée pour les protocoles POP, IMAP et SMTP. 
  
```XML
<EncryptionMethod/>
```

 **string**
## <a name="attributes-and-elements"></a>Attributs et éléments

Les sections suivantes décrivent les attributs, éléments enfants et éléments parents.
  
### <a name="attributes"></a>Attributs

Aucun.
  
### <a name="child-elements"></a>Éléments enfants

Aucun.
  
### <a name="parent-elements"></a>Éléments parents

|**Élément**|**Description**|
|:-----|:-----|
|[ProtocolConnection (SOAP)](protocolconnection-soap.md) <br/> |Représente le protocole de connexion du client Web server.  <br/> |
   
## <a name="text-value"></a>Valeur de texte

La valeur de texte pour cet élément est la méthode de chiffrement qui est utilisée pour les protocoles POP, IMAP et SMTP.
