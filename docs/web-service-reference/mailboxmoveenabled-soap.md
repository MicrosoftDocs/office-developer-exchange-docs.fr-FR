# <a name="mailboxmoveenabled-soap"></a>MailboxMoveEnabled (SOAP)

L’élément **MailBoxMoveEnabled** représente l’indicateur **MailboxMoveEnabled()** . L’élément **MailBoxMoveEnabled** est à usage interne uniquement. Cet élément n’est pas utilisé par les clients. 
  
```XML
<MailBoxMoveEnabled>true | false</MailBoxMoveEnabled>
```

**Boolean**

## <a name="attributes-and-elements"></a>Attributs et éléments

Les sections suivantes décrivent les attributs, éléments enfants et éléments parents.
  
### <a name="attributes"></a>Attributs

### <a name="child-elements"></a>Éléments enfants
