# <a name="mailboxmoveenabled-soap"></a>MailboxMoveEnabled (SOAP)

L’élément **MailBoxMoveEnabled** représente l’indicateur **MailBoxMoveEnabled ()** . L’élément **MailBoxMoveEnabled** est réservé à un usage interne. Cet élément n’est pas utilisé par les clients. 
  
```XML
<MailBoxMoveEnabled>true | false</MailBoxMoveEnabled>
```

**Boolean**

## <a name="attributes-and-elements"></a>Attributs et éléments

Les sections suivantes décrivent les attributs, les éléments enfants et les éléments parents.

### <a name="child-elements"></a>Éléments enfants
