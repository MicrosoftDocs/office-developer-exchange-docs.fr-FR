# <a name="errorcode-pox"></a>ErrorCode (POX)

L’élément **ErrorCode** contient le code d’erreur pour une erreur de réponse de découverte automatique. 
  
[Erreur (POX)](error-pox.md)
  
[ErrorCode (POX)](errorcode-pox.md)
  
```xml
<ErrorCode/>
```

## <a name="attributes-and-elements"></a>Attributs et éléments

Les sections suivantes décrivent les attributs, éléments enfants et éléments parents.
  
### <a name="attributes"></a>Attributs

Aucun.
  
### <a name="child-elements"></a>Éléments enfants

Aucun.
  
### <a name="parent-elements"></a>Éléments parents

|**Élément**|**Description**|
|:-----|:-----|
|[Erreur (POX)](error-pox.md) <br/> |Contient une réponse d’erreur de découverte automatique.  <br/> |
   
## <a name="text-value"></a>Valeur de texte

La valeur de texte représente le code d’erreur pour une erreur de réponse de découverte automatique.
