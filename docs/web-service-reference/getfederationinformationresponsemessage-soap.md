# <a name="getfederationinformationresponsemessage-soap"></a>GetFederationInformationResponseMessage (SOAP)

L’élément **GetFederationInformationResponseMessage** définit une réponse à une demande [SOAP (GetFederationInformation Operation)](getfederationinformation-operation-soap.md) . 
  
```XML
<GetFederationInformationResponseMessage>
   <Response/>
</GetFederationInformationResponseMessage>
```

 **GetFederationInformationResponseMessage**
