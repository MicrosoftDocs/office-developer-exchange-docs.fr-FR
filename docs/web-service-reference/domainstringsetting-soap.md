# <a name="domainstringsetting-soap"></a>DomainStringSetting (SOAP)

L’élément **DomainStringSetting** représente un paramètre du domaine dont la valeur est de type chaîne. 
  
```XML
<DomainStringSetting>
   <Name/>
   <Value/>
</DomainStringSetting>
```

 **DomainStringSetting**
## <a name="attributes-and-elements"></a>Attributs et éléments

Les sections suivantes décrivent les attributs, éléments enfants et éléments parents.
  
### <a name="attributes"></a>Attributs

Aucun.

|[Valeur (SOAP)](value-soap.md) <br/> |Représente une valeur de paramètre utilisateur.  <br/> |
