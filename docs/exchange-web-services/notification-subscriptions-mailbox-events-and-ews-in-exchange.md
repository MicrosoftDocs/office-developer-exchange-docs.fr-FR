## <a name="how-do-ews-notifications-work"></a><span data-ttu-id="8bb85-161">Comment fonctionnent les notifications EWS ?</span><span class="sxs-lookup"><span data-stu-id="8bb85-161">How do EWS notifications work?</span></span>

<span data-ttu-id="8bb85-p112">Les notifications EWS sont gérées par abonnement.
