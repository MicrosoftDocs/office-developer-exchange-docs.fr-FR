## <a name="to-use-the-ews-managed-api-to-communicate-with-ews"></a><span data-ttu-id="aacad-146">Pour utiliser l’API managée EWS pour communiquer avec EWS</span><span class="sxs-lookup"><span data-stu-id="aacad-146">To use the EWS Managed API to communicate with EWS</span></span>

1. <span data-ttu-id="aacad-147">Instanciez la classe **ExchangeService** .</span><span class="sxs-lookup"><span data-stu-id="aacad-147">Instantiate the **ExchangeService** class.</span></span> 
    
   ```csharp
    ExchangeService service = new ExchangeService();
   ```

   > [!NOTE]
   > <span data-ttu-id="aacad-148">L’instanciation de **ExchangeService** avec un constructeur vide crée une instance liée à la dernière version connue d’Exchange.</span><span class="sxs-lookup"><span data-stu-id="aacad-148">Instantiating **ExchangeService** with an empty constructor will create an instance that is bound to the latest known version of Exchange.</span></span> <span data-ttu-id="aacad-149">Vous pouvez également cibler une version spécifique d’Exchange en spécifiant version comme paramètre.</span><span class="sxs-lookup"><span data-stu-id="aacad-149">Alternatively, you can target a specific version of Exchange by specifying version as a parameter.</span></span> `ExchangeService service = new ExchangeService(ExchangeVersion.Exchange2007_SP1);`
  
2. <span data-ttu-id="aacad-150">Définissez les informations d’identification de l’utilisateur qui envoie des demandes au serveur Exchange.</span><span class="sxs-lookup"><span data-stu-id="aacad-150">Set the credentials of the user who sends requests to the Exchange server.</span></span> <span data-ttu-id="aacad-151">Si vous souhaitez vous connecter à EWS à partir d’un ordinateur connecté au domaine, à l’aide des informations d’identification de l’utilisateur authentifié, définissez la propriété **UseDefaultCredentials** de l’objet **ExchangeService** sur **true**.</span><span class="sxs-lookup"><span data-stu-id="aacad-151">If you want to connect to EWS from a computer that is logged on to the domain, using the credentials of the authenticated user, set the **UseDefaultCredentials** property on the **ExchangeService** object to **true**.</span></span>
    
   ```cs
    // Connect by using the default credentials of the authenticated user.
    service.UseDefaultCredentials = true;
   ```

   <span data-ttu-id="aacad-152">Si vous ne souhaitez pas vous connecter à l’aide des informations d’identification par défaut de l’utilisateur, définissez la propriété **Credentials** sur l’objet **ExchangeService** afin de spécifier explicitement les informations d’identification d’un autre utilisateur.</span><span class="sxs-lookup"><span data-stu-id="aacad-152">If you do not want to connect by using the default user credentials, set the **Credentials** property on the **ExchangeService** object to explicitly specify the credentials of a different user.</span></span> <span data-ttu-id="aacad-153">Si vous utilisez Exchange Online ou Exchange Online dans le cadre d’Office 365, vous utiliserez l’authentification de base, avec un nom d’utilisateur et un mot de passe.</span><span class="sxs-lookup"><span data-stu-id="aacad-153">If you are using Exchange Online or Exchange Online as part of Office 365, you'll use basic authentication, with just a user name and password.</span></span> <span data-ttu-id="aacad-154">Un nom de domaine est requis pour l’authentification NTLM.</span><span class="sxs-lookup"><span data-stu-id="aacad-154">A domain name is required for NTLM authentication.</span></span>

   ```

   <span data-ttu-id="aacad-155">Vous pouvez également spécifier les informations d’identification de l’utilisateur à l’aide du nom de domaine et du mot de passe de l’utilisateur.</span><span class="sxs-lookup"><span data-stu-id="aacad-155">You can also specify the credentials of the user by using the user's domain name and password.</span></span>

   ```

   > [!NOTE]
   > <span data-ttu-id="aacad-156">Si la propriété **UseDefaultCredentials** est définie sur **true**, la valeur de la propriété **Credentials** est ignorée.</span><span class="sxs-lookup"><span data-stu-id="aacad-156">If the **UseDefaultCredentials** property is set to **true**, the value of the **Credentials** property is ignored.</span></span>
